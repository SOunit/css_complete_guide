flex box (default)

default is

  1. flex-direction: row;
  2. flex-wrap: nowrap;
  3. justify-content: flex-start;
  4. align-items: stretch;

important theory is

  1. main-axis (same with flex-direction)
  2. cross-axis (opposite of main-axis)

important properties is

  1. justify-content is for main-axis
  2. align-items is for cross-axis
