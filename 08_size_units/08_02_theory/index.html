3 rules to remember

1. position: fixed

% reference to viewport

2. position: absolute

% reference to content and padding of ancestor with position active ( not position static ). In case it find no ancestor with position active, then reference to viewport, same with position fixed.

3. position: static / relative

% reference to ancestor content with block level.
